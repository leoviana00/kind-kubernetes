# Extra Port Mappings

## Ajustando as configuraÃ§Ãµes do Kind

- Adicionando configuraÃ§Ã£o do bind para acessar a api

```
kind: Cluster
apiVersion: kind.x-k8s.io/v1alpha4
nodes:
- role: control-plane
  extraPortMappings:
  - containerPort: 30001
    hostPort: 8080
    listenAddress: "0.0.0.0"
    protocol: TCP  
- role: worker
- role: worker
```
